- Проектирование и внедрение LLM‑решений: чат‑боты, ассистенты, RAG‑системы
- Интеграция LLM с бизнес‑процессами: CRM, HelpDesk, база знаний, документооборот
- Разработка ботов (Telegram и др.): от прототипа до продакшена
- Автоматизация и оркестрация: n8n, Make, API‑интеграции
- Классическая инфраструктура: VMware, Windows/Linux, SQL Server, AD, backup, monitoring
